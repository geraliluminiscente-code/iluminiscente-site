2. Dados pessoais recolhidos

Podem ser recolhidos os seguintes dados pessoais:

  • Conteúdo das mensagens e avaliações submetidas

3. Finalidade do tratamento

Os dados pessoais são utilizados exclusivamente para:

4. Fundamento legal

O tratamento dos dados baseia-se no consentimento do titular dos dados, prestado aquando do envio dos formulários existentes no site.

5. Conservação dos dados

Os dados pessoais são conservados apenas durante o período necessário para a finalidade a que se destinam ou enquanto existir obrigação legal.

6. Partilha de dados

Os dados não são partilhados com terceiros, exceto quando necessário para cumprimento de obrigação legal.

Os formulários do site utilizam o serviço Formspree, que assegura conformidade com o RGPD.

7. Direitos do titular dos dados

Nos termos da legislação aplicável, o titular dos dados tem direito a:

  • Acesso aos seus dados pessoais
  • Retificação de dados inexatos ou incompletos
  • Apagamento dos dados
  • Limitação ou oposição ao tratamento

O exercício destes direitos pode ser efetuado através do email indicado.

8. Alterações à política

A Iluminiscente reserva-se o direito de atualizar a presente Política de Privacidade sempre que necessário.
